  Markdown是一种轻量级标记语言,创始人为John Gruber. Markdown对于图片,图表和数学有相应的支持.

标题

  Markdown使用 # 符号表示标题标签.不同数量表示不同级别的标题.

# 一级标题

## 二级标题

### 三级标题

样式

  Markdown可使用特殊调节语法字体样式,包括倾斜,加粗等。

`突出`  **加粗** *倾斜*  ==标记== --删除-- > 引用

文本

文本框用两个 ``` 表示,文本框中间的文字将不使用Markdown语法解析.采用Highlight则需要在第一组三个点之后加上需要高亮的语法.

int main() {
    puts("hello");
}
